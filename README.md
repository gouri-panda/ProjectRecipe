# Recipe Search App

A simple Android recipe search app using Spoonacular API and Jetpack Compose.

## Features

- Fetch and display popular and all recipes from Spoonacular API
- View complete recipe details including ingredients, instructions, and similar recipes
- Search for recipes and mark as favorite
- Bottom navigation for Home and Favorites
- Local caching of favorite recipes using Room Database
- Custom search bar with a unique shape
